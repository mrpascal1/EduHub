1. Debian:

  • Debian has its existence since 1993 and releases its versions much slowly then Ubuntu and Linux mint.
  • Commonly used on server and desktop computer.
  • This makes it one of the most stable Linux distributor.

2. SUSE:

  • SUSE Linux Enterprise Server is upstream for openSUSE.
  • Linux Kernel 3.11 used in openSUSE 12.3.
  • SUSE system used in retail sectors.

3. Fedora:

  • It doesn't make its own desktop environment but used 'upstream' software.
  • Fedora is upstream for CentOS, RHEL and Oracle Linux.
  • It is less stable but provides the latest stuff.

4. Ubuntu:

  • It is the most well known Linux distribution.
  • Ubuntu is a next version of Debian and easy to use for newbies.
  • It comes with a lots of pre-installed apps and easy to use repositories libraries.

5. Red Hat Enterprise / CentOS:

  • Red hat is a commercial Linux distributor.
  • RHEL used by enterprises which host their own system.
  • Red hat uses trademark law to prevent their software from being redistributed. 
  • CentOS  it is a free version of RHEL and provide a stable platform for a long time.
