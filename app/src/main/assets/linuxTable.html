Name Function
Kernel It connects the application software to the hardware of a computer. For e.g. : Linux Kernel
Distribution It's a collection of application which is combined to make an operating system. For e.g. : Ubuntu
Bootloader It's a program that puts the operating system into Memory. For e.g. : GRUB
Service It's a program that runs on background process. For e.g. : http, ftpd
File System It's a method for storing and organizing of files. For e.g. : ext3 (3rd Extended filesystem), FAT (File Allocation Table)
Desktop Environment It provides GUI (Graphical User Interface) on the operating system. For e.g. : GNOME (GNU Network Object Model Environment)
Command Linet It's an interface for typing commands on top of operating system.
Shell It interprets The command line input and instructs operating system to perform tasks and commands. For e.g. : bash
